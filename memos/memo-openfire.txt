
API Rest: Intaller le plugin puis aller dans Serveur > Paramètres du serveur > REST API
    Doc: https://www.igniterealtime.org/projects/openfire/plugins/restapi/readme.html#feature-list
    Adresse: HTTPS: var apiBaseUrl = "https://test-messagerie.ddns.net:9091/plugins/restapi/v1";
             HTTPS: var apiBaseUrl = "http://test-messagerie.ddns.net:9090/plugins/restapi/v1";

Websocket disponible après activation du plugin nécéssaire.
    Port d'écoute: 7443

Voir l'option d'audit de messages permettant de conserver certains type de messages pour analyse
    Serveur > Paramètres du serveur > Audit de message

Voir l'option Clustering permettant de supporter une plus grande charge
    Serveur > Gestion du serveur > Clustering

Voir l'option Média proxy permettant de faire transiter des données si les connexions paires à paires échouent
    Serveur > Media services > Media proxy
